audit_sys_operations 
=========================================

audit_sys_operations specifies whether to track the operations of system users. 


|          **Property**           |                                            **Description**                                             |
|---------------------------------|--------------------------------------------------------------------------------------------------------|
| Parameter type                  | Boolean                                                                                                |
| Default value                   | False                                                                                                  |
| Value range                     | * False   * True    |
| Effective upon OBServer restart | No                                                                                                     |

audit_trail 
================================

audit_trail specifies whether to enable database auditing. 


|            Property             |                                                                                                                  Description                                                                                                                   |
|---------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Parameter type                  | String                                                                                                                                                                                                                                         |
| Default value                   | None                                                                                                                                                                                                                                           |
| Value range                     | * None   * OS   * DB   * DB, EXTENDED   * DB_EXTENDED    |
| Effective upon OBServer restart | No                                                                                                                                                                                                                                             |

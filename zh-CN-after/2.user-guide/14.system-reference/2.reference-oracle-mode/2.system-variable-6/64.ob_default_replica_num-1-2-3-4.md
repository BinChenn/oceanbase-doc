ob_default_replica_num 
===========================================

ob_default_replica_num 用于设置每个集群中表的默认副本数。
**说明**



当前变量已不再使用。


| **属性** |                                                   **描述**                                                   |
|--------|------------------------------------------------------------------------------------------------------------|
| 参数类型   | int                                                                                                        |
| 默认值    | 1                                                                                                          |
| 取值范围   | \[1, 7\]                                                                                                   |
| 生效范围   | * GLOBAL   * SESSION    |
| 使用状态   | 已废弃                                                                                                        |

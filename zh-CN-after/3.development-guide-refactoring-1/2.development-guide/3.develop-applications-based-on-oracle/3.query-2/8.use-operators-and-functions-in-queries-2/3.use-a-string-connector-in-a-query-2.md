查询中使用字符串连接符 
================================

字符串连接符，Oracle 租户通常使用 \`\|\|\` 作为字符串连接符。

**示例** 
---------------------------

"\|\|"作为字符串连接符

```unknow
obclient> SELECT c_first || ' ' || c_last  full_name FROM cust ORDER BY c_last;
+-------------------+
| FULL_NAME         |
+-------------------+
| Michael Brown     |
| Michael Brown     |
| Michael Brown     |
| Michael Brown     |
| Madeleine Johnson |
| Madeleine Johnson |
| Madeleine Johnson |
| Madeleine Johnson |
| Ann Smith         |
| Ann Smith         |
| Ann Smith         |
| Ann Smith         |
+-------------------+
12 rows in set
```

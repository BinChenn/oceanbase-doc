TRANSFORM 
==============================

TRANSFORM 成员函数使用 XSL 样式表参数和顶级参数（name=value）转换 XML 数据。如果除 `parammap` 之外的任一参数为 NULL，则返回 NULL。

语法 
-----------------------

```unknow
MEMBER FUNCTION TRANSFORM(
    xsl IN XMLType)
RETURN XMLType;
```



参数解释 
-------------------------



|    参数    | IN / OUT |            解释            |
|----------|----------|--------------------------|
| xsl      | (IN)     | 描述转换的 XSL 样式表。           |
| parammap | (IN)     | XSL 的顶级参数，即`name=value`。 |

nls_sort 
=============================

`nls_sort` 用于查看字符串值的排序规则。


|  **属性**  |                                                   **描述**                                                   |
|----------|------------------------------------------------------------------------------------------------------------|
| 参数类型     | varchar                                                                                                    |
| 默认值      | BINARY                                                                                                     |
| 取值范围     | N/A                                                                                                        |
| 生效范围     | * Global   * Session    |
| 是否参与序列化  | 是                                                                                                          |
| 是否影响计划生成 | 是                                                                                                          |


**说明**



该变量当前暂不生效。

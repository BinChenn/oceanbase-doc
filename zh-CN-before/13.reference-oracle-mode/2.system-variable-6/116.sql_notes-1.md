sql_notes 
==============================

`sql_notes` 用于设置


|    属性    |                                                     描述                                                     |
|----------|------------------------------------------------------------------------------------------------------------|
| 参数类型     | bool                                                                                                       |
| 默认值      | 0                                                                                                          |
| 取值范围     | * 0：表示   * 1：表示         |
| 生效范围     | * Global   * Session    |
| 是否参与序列化  | 否                                                                                                          |
| 是否影响计划生成 | 否                                                                                                          |
| 是否可以为空   | 否                                                                                                          |
| 是否可修改    | 是                                                                                                          |

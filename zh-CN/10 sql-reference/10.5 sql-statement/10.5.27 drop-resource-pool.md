DROP RESOURCE POOL 
=======================================



描述 {#OD2e2}
-----------

该语句用来删除资源池。

格式 {#psOkd}
-----------

    DROP RESOURCE POOL poolname; 



参数解释 {#cSmuf}
-------------



|  **参数**  |  **描述**   |
|----------|-----------|
| poolname | 要删除的资源池名称 |



示例 {#OA5e4}
-----------

* 删除资源池 pool1。




    OceanBase(admin@test)> DROP RESOURCE POOL pool1;
    Query OK, 0 rows affected (0.00 sec)

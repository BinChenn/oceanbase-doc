DROP RESOURCE UNIT 
=======================================



描述 {#skyzN}
-----------

该语句用来删除资源单元配置。

格式 {#5K75U}
-----------

    DROP RESOURCE UNIT unitname 



参数解释 {#1YpbA}
-------------



|  **参数**  |    **描述**     |
|----------|---------------|
| unitname | 要删除的资源单元配置的名称 |





示例 {#UBaY0}
-----------

* 删除资源单元配置unit1。




    OceanBase(admin@test)> DROP RESOURCE UNIT unit1;
    Query OK, 0 rows affected (0.00 sec)

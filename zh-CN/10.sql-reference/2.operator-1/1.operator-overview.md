运算符概述 
==========================

运算符一般用于连接操作数或参数等的单个数据项且返回结果。从语法上讲，运算符可以出现在操作数之前、操作数之后或两个操作数之间。

常见的运算符还可以分为一元和二元运算符:

* 一元运算符

  一元运算符仅对一个操作数进行运算。一元运算符常用的格式如下：

  ```javascript
  运算符 运算数
  ```

  




<!-- -->

* 二元运算符

  二元运算符是对两个操作数进行运算。二元运算符的常用格式：

  ```javascript
  运算数1 运算符 运算数2
  ```

  




其他具有特殊格式的运算符可接受两个以上的操作数。如果为运算符提供了空操作数 `NULL`， 则结果始终为空。唯一不遵循此规则的是字符连接运算符（\|\|）。

LOWER 
==========================



`LOWER` 函数将字符串全部转为小写。

语法 {#f2b0b493}
--------------

    LOWER(c1)



参数 {#3d0a2df9}
--------------



| 参数 |                              说明                               |
|----|---------------------------------------------------------------|
| c1 | 表示字符串。字符串类型可为：`CHAR`、`VARCHAR2`、`NCHAR`、`NVARCHAR2` 或 `CLOB`。 |



返回类型 {#322b25e7}
----------------

返回与 `c1` 相同的数据类型 。

示例 {#1a63ac23}
--------------

执行以下语句：

    SELECT  lower('AaBbCcDd')AaBbCcDd FROM DUAL;



查询结果如下：

    +----------+
    | AABBCCDD |
    +----------+
    | aabbccdd |
    +----------+

PURGE RECYCLEBIN 
=====================================



描述 {#a2UmD}
-----------

该语句用于 root 用户清空回收站。
**注意**



PURGE RECYCLEBIN会清除回收站中的所有东西，只有 root 用户才能够执行这条语句，操作时请谨慎。

格式 {#82oTU}
-----------

     PURGE RECYCLEBIN;



参数解释 {#xUyCI}
-------------

无。

示例 {#o8uod}
-----------

* root 用户清空回收站。

  




    OceanBase(admin@test)> purge recyclebin;
    Query OK, 0 rows affected (0.03 sec)

DROP RESTORE POINT 
=======================================



描述 
-----------------------

该语句用来删除还原点（Restore Point）。

语法 
-----------------------

```sql
DROP RESTORE POINT restore_point_name;
```



参数解释 
-------------------------



|         参数         |    描述    |
|--------------------|----------|
| restore_point_name | 指定还原点名称。 |



示例 
-----------------------

删除一个还原点 `rp1`。

```sql
obclient> DROP RESTORE POINT rp1;
Query OK, 0 rows affected
```

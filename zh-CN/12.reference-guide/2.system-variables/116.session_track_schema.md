session_track_schema 
=========================================

`session_track_schema` 用于控制返回的 OK 包中是否包含 schema 变更的信息。


| **属性** |                                                   **描述**                                                   |
|--------|------------------------------------------------------------------------------------------------------------|
| 参数类型   | bool                                                                                                       |
| 默认值    | 1                                                                                                          |
| 取值范围   | * 0   * 1               |
| 生效范围   | * Global   * Session    |

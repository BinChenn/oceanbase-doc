session_track_state_change 
===============================================

`session_track_state_change` 用于控制返回的 OK 包中是否包含 Session 状态变更的信息。


| **属性** |                                                   **描述**                                                   |
|--------|------------------------------------------------------------------------------------------------------------|
| 参数类型   | bool                                                                                                       |
| 默认值    | 0                                                                                                          |
| 取值范围   | * 0   * 1               |
| 生效范围   | * Global   * Session    |

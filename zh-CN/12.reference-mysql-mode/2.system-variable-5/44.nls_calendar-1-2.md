nls_calendar 
=================================

`nls_calendar` 用于控制数据库使用的日历系统。


| **属性**  |                                                   **描述**                                                   |
|---------|------------------------------------------------------------------------------------------------------------|
| 参数类型    | varchar                                                                                                    |
| 默认值     | GREGORIAN                                                                                                  |
| 取值范围    | N/A                                                                                                        |
| 生效范围    | * Global   * Session    |
| 是否参与序列化 | 是                                                                                                          |


**说明**



该变量当前暂不生效。

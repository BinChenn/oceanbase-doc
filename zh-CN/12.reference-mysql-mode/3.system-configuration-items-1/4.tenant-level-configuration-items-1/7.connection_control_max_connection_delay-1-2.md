connection_control_max_connection_delay 
============================================================

`connection_control_max_connection_delay` 指定了超过错误登录次数阈值之后，错误登录锁定的最大时长。


|        属性        |         描述          |
|------------------|---------------------|
| 参数类型             | 整型                  |
| 默认值              | 2147483647，单位为毫秒    |
| 取值范围             | \[1000，2147483647\] |
| 是否重启 OBServer 生效 | 否                   |

system_time_zone 
=====================================

`system_time_zone` 用于设置服务器机器系统时区。


| **属性**  |                                                              **描述**                                                              |
|---------|----------------------------------------------------------------------------------------------------------------------------------|
| 参数类型    | varchar                                                                                                                          |
| 默认值     | CST                                                                                                                              |
| 取值范围    | * 数值范围：\[-12:59,+13:00\]   * CDT、CST 等时区地域    |
| 生效范围    | Global                                                                                                                           |
| 是否可修改   | 该变量支持在创建租户时指定，租户创建后不能通过 SQL 语句修改。具体设置请参见《管理员指南》的 [设置变量](/zh-CN/5.administrator-guide/2.basic-database-management/2.configuration-management/3.set-variables.md) 章节                          |
| 是否参与序列化 | 是                                                                                                                                |

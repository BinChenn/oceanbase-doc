ACTIVE_INSTANCES 
=====================================

`ACTIVE_INSTANCES` 存储过程用于返回激活的实例。

语法 
-----------------------

```unknow
DBMS_UTILITY.ACTIVE_INSTANCES (
instance_table   OUT INSTANCE_TABLE,
instance_count   OUT NUMBER);
```



参数解释 
-------------------------



|       参数       |               解释                |
|----------------|---------------------------------|
| instance_table | 包含激活实例编号和名称的列表。如果当前没有启动实例，列表为空。 |
| instance_count | 激活实例的数量。                        |

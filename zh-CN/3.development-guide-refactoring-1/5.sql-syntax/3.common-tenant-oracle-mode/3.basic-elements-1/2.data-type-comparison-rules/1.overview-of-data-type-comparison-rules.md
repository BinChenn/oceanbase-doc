数据类型比较规则概述 
===============================

OceanBase 数据库支持如下数据类型比较规则：

* [日期值比较规则](../2.data-type-comparison-rules/3.date-value.md)

  

* [字符值比较规则](../2.data-type-comparison-rules/4.character-value.md)

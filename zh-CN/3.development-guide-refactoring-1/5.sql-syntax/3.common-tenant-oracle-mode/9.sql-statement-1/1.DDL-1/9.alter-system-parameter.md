ALTER SYSTEM PARAMETER 
===========================================



描述 
-----------------------

该语句用来修改本租户的配置项信息。

语法 
-----------------------

```sql
ALTER SYSTEM SET parameter_name = expression;
```



参数解释 
-------------------------



|     **参数**     |    **描述**    |
|----------------|--------------|
| parameter_name | 指定要修改的配置项名称。 |
| expression     | 指定修改后的配置项的值。 |

-----------------------

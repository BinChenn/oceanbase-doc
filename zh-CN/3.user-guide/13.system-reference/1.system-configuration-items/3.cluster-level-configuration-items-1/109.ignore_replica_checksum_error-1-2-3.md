ignore_replica_checksum_error 
==================================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/ignore_replica_checksum_error-1-2-3
ignore_replica_checksum_error 用于设置是否忽略多副本间校验和检查发生的错误。


|      **属性**      |                                                 **描述**                                                 |
|------------------|--------------------------------------------------------------------------------------------------------|
| 参数类型             | 布尔类型                                                                                                   |
| 默认值              | False                                                                                                  |
| 取值范围             | * True   * False    |
| 是否重启 OBServer 生效 | 否                                                                                                      |

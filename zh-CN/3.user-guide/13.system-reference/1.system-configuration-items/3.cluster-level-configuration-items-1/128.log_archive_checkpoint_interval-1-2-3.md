log_archive_checkpoint_interval 
====================================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/log_archive_checkpoint_interval-1-2-3
`log_archive_checkpoint_interval` 用于设置冷数据的日志归档检查点的时间间隔。


|      **属性**      |   **描述**   |
|------------------|------------|
| 参数类型             | 时间类型       |
| 默认值              | 120s       |
| 取值范围             | \[5s, 1h\] |
| 是否重启 OBServer 生效 | 否          |

merger_warm_up_duration_time 
=================================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/merger_warm_up_duration_time-1-2-3
merger_warm_up_duration_time 用于设置合并时，新版基线数据的预热时间。


|      **属性**      |   **描述**    |
|------------------|-------------|
| 参数类型             | 时间类型        |
| 默认值              | 0s          |
| 取值范围             | \[0s, 60m\] |
| 是否重启 OBServer 生效 | 否           |

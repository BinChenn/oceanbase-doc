migrate_concurrency 
========================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/migrate_concurrency-1-2-3
migrate_concurrency 用于设置内部数据迁移的最大并发数。


|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 整型        |
| 默认值              | 10        |
| 取值范围             | \[0, 64\] |
| 是否重启 OBServer 生效 | 否         |

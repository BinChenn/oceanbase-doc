multiblock_read_size 
=========================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/multiblock_read_size-1-2-3
multiblock_read_size 用于设置读取数据时 IO 聚合大小。


|      **属性**      |   **描述**   |
|------------------|------------|
| 参数类型             | 容量单位       |
| 默认值              | 128k       |
| 取值范围             | \[0K, 2M\] |
| 是否重启 OBServer 生效 | 否          |

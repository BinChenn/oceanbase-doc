row_purge_thread_count 
===========================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/row_purge_thread_count-1-2-3
row_purge_thread_count 用于设置执行内存中行内数据合并的最大工作线程数。


|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 整型        |
| 默认值              | 4         |
| 取值范围             | \[1, 64\] |
| 是否重启 OBServer 生效 | 否         |

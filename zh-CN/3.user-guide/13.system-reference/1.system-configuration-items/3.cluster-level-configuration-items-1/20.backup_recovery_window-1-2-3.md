# backup_recovery_window 

`backup_recovery_window` 用于设置备份数据可以恢复的时间。

## 描述 



|      **属性**      |  **描述**  |
|------------------|----------|
| 参数类型             | 时间类型     |
| 默认值              | 0        |
| 取值范围             | \[0, +∞) |
| 是否重启 OBServer 生效 | 否        |


> **注意**<br>
> * 该配置项的取值单位为天（d），设置取值时，需带上单位才能生效。
> * 默认值为 0，表示不自动清理。

## 示例 

```sql
obclient> ALTER SYSTEM SET backup_recovery_window = '7d';
```

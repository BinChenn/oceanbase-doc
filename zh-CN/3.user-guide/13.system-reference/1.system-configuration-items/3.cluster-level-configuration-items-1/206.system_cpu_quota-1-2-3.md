system_cpu_quota 
=====================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/system_cpu_quota-1-2-3
system_cpu_quota 用于设置系统租户可使用的 CPU 配额。


|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 双精度浮点数    |
| 默认值              | 10        |
| 取值范围             | \[0, 16\] |
| 是否重启 OBServer 生效 | 否         |

zone_merge_order 
=====================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/zone_merge_order-1-2-3
zone_merge_order 用于设置 Zone 的轮转合并顺序。不指定时，由系统自动决定。


|      **属性**      | **描述** |
|------------------|--------|
| 参数类型             | 字符串类型  |
| 默认值              | 空值     |
| 取值范围             | NULL   |
| 是否重启 OBServer 生效 | 否      |

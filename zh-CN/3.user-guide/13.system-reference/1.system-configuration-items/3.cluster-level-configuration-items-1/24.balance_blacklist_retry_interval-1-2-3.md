balance_blacklist_retry_interval 
=====================================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/balance_blacklist_retry_interval-1-2-3
balance_blacklist_retry_interval 用于设置副本迁移等后台任务被放入黑名单后的重试时间间隔。


|      **属性**      |    **描述**    |
|------------------|--------------|
| 参数类型             | 时间类型         |
| 默认值              | 30m，表示 30 分钟 |
| 取值范围             | \[0s, 180m\] |
| 是否重启 OBServer 生效 | 否            |

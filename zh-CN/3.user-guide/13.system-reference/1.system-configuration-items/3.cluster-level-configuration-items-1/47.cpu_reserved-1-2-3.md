cpu_reserved 
=================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/cpu_reserved-1-2-3
cpu_reserved 用于设置系统预留的 CPU 数量，其余 CPU 将被 OceanBase 数据库独占使用。


|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 整型        |
| 默认值              | 2         |
| 取值范围             | \[0, 15\] |
| 是否重启 OBServer 生效 | 否         |

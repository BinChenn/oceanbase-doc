enable_auto_leader_switch 
==============================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/enable_auto_leader_switch-1-2
enable_auto_leader_switch 用于设置是否开启自动切主。


|      **属性**      |                                                 **描述**                                                 |
|------------------|--------------------------------------------------------------------------------------------------------|
| 参数类型             | 布尔类型                                                                                                   |
| 默认值              | True                                                                                                   |
| 取值范围             | * True   * False    |
| 是否重启 OBServer 生效 | 否                                                                                                      |

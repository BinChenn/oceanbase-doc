enable_separate_sys_clog 
=============================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/enable_separate_sys_clog-1-2-3
enable_separate_sys_clog 用于设置是否把系统事务日志与用户事务日志分开存储。


|      **属性**      |                                                 **描述**                                                 |
|------------------|--------------------------------------------------------------------------------------------------------|
| 参数类型             | 布尔类型                                                                                                   |
| 默认值              | False                                                                                                  |
| 取值范围             | * True   * False    |
| 是否重启 OBServer 生效 | 否                                                                                                      |

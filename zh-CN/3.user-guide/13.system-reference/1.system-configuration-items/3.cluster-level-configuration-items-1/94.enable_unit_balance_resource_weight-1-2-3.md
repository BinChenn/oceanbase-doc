enable_unit_balance_resource_weight 
========================================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/enable_unit_balance_resource_weight-1-2-3
enable_unit_balance_resource_weight 用来设置是否开启手动配置资源权重的功能。
**说明**



在当前版本中，该配置项已不再使用。


|      **属性**      |     描述     |
|------------------|------------|
| 参数类型             | boolean 类型 |
| 默认值              | false      |
| 取值范围             | NULL       |
| 是否重启 OBServer 生效 | 否          |

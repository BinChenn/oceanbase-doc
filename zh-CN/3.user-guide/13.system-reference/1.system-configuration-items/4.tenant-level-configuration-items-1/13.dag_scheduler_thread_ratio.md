dag_scheduler_thread_ratio 
===============================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/dag_scheduler_thread_ratio
`dag_scheduler_thread_ratio` 用于设置 dag 调度器所有工作线程占用 CPU 时间片的比例。


|      **属性**      |  **描述**  |
|------------------|----------|
| 参数类型             | 整型       |
| 默认值              | 0        |
| 取值范围             | \[0, 1000\] |
| 是否重启 OBServer 生效 | 否        |

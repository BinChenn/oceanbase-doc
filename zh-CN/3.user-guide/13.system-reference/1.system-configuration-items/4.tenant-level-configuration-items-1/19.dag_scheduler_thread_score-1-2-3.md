major_merge_thread_score 
=============================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/dag_scheduler_thread_score-1-2-3
`major_merge_thread_score` 用于设置 major compaction 的工作线程占用时间片权重。


|      **属性**      |  **描述**  |
|------------------|----------|
| 参数类型             | 整型       |
| 默认值              | 0        |
| 取值范围             | \[0,64\] |
| 是否重启 OBServer 生效 | 否        |

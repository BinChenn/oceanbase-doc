character_set_client 
=========================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/character_set_client-1-2-3-4
character_set_client 用于设置客户端发送语句的字符集。


| **属性**  |                                                                       **描述**                                                                        |
|---------|-----------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型    | string                                                                                                                                              |
| 默认值     | utf8mb4                                                                                                                                             |
| 取值范围    | * utf8mb4   * binary   * gbk    |
| 生效范围    | * GLOBAL   * SESSION                                             |
| 是否参与序列化 | 是                                                                                                                                                   |
| 是否可以为空  | 是                                                                                                                                                   |

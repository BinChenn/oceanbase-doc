oceanbase.DBA_OB_DATABASES 
===============================================



功能 
-------------------

展示租户的数据库元信息。

字段说明 
----------------------



|     字段名称      |      类型       | 是否可以为 NULL |             描述             |
|---------------|---------------|------------|----------------------------|
| DATABASE_NAME | varchar(128)  | NO         | 数据库名                       |
| IN_RECYCLEBIN | varchar(3)    | NO         | 数据库是否在回收站中，值为 `YES` 或 `NO` |
| COLLATION     | varchar(128)  | NO         | 数据库缺省 COLLATION            |
| COMMENT       | varchar(2048) | NO         | 数据库的描述                     |

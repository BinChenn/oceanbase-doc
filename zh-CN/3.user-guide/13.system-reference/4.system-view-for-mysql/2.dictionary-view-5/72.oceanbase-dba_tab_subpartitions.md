oceanbase.DBA_TAB_SUBPARTITIONS 
====================================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/oceanbase-dba_tab_subpartitions


功能 
--------------------

展示子分区名称、表和所属分区的名称、其存储属性以及 `DBMS_STATS` 包生成的统计信息。

TENANT_VIRTUAL_SESSION_VARIABLE 
====================================================



功能 
-----------

记录 Session 的变量信息。

字段说明 
-------------



|   **字段名称**    |     **类型**      | **是否可以为 NULL** |   **描述**    |
|---------------|-----------------|----------------|-------------|
| VARIABLE_NAME | VARCHAR2(128)   | NO             | Session 变量名 |
| VALUE         | VARCHAR2(65536) | YES            | Session 变量值 |

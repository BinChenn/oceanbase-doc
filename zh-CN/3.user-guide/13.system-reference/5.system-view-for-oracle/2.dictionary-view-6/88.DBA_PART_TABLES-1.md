DBA_PART_TABLES 
====================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/DBA_PART_TABLES-1


功能 
-----------

查看当前用户可访问的分区表的分区信息。

相关视图 
-------------

* `ALL_PART_TABLES`：查看所有的分区表的分区信息

  

* `USER_PART_TABLES`：查看属于当前用户的分区表的分区信息

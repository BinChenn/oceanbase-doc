# GV$SYSSTAT/V$SYSSTAT
## 功能
MySQL 模式
GV$SYSSTAT 展示所有服务器上租户级别的统计事件信息。
V$SYSSTAT 展示当前服务器上租户级别的统计事件信息。
Oracle 模式
GV$SYSSTAT 展示所有服务器上系统级别的统计事件信息。
其中：

- STATISTIC#：表示统计项编号，同一统计项的编号在不同 OBServer 版本中可能会发生变化。
- STAT_ID：表示统计项 ID，同一统计项的 ID 在不同 OBServer 版本中保证相同。

V$SYSSTAT 展示当前服务器上系统级别的统计事件信息。
其中：

- STATISTIC#：表示统计项编号，同一统计项的编号在不同 OBServer 版本中可能会发生变化。
- STAT_ID：表示统计项 ID，同一统计项的 ID 在不同 OBServer 版本中保证相同。
## 字段说明
MySQL 模式

| **字段名称** | **类型** | **是否可以为 NULL** | **描述** |
| --- | --- | --- | --- |
| CON_ID | bigint(20) | NO | 租户 ID |
| SVR_IP | varchar(46) | NO | 信息所在服务器的 IP 地址 |
| SVR_PORT | bigint(20) | NO | 信息所在服务器的端口号 |
| STATISTIC# | bigint(20) | NO | 统计事件的下标 |
| NAME | varchar(64) | NO | 统计事件的名称 |
| CLASS | bigint(20) | NO | 统计事件所属的类型 |
| VALUE | bigint(20) | NO | 统计事件对应的结果数值 |
| VALUE_TYPE | varchar(16) | NO | 统计值类型：
- 累加
- 更新
 |
| STAT_ID | bigint(20) | NO | 统计事件 ID |

Oracle 模式

| **字段名称** | **类型** | **是否可以为 NULL** | **描述** |
| --- | --- | --- | --- |
| CON_ID | NUMBER | NO | 租户 ID |
| SVR_IP | VARCHAR2(46) | NO | 信息所在服务器的 IP 地址 |
| SVR_PORT | NUMBER(38) | NO | 信息所在服务器的端口号 |
| STATISTIC# | NUMBER | NO | 统计事件的下标 |
| NAME | VARCHAR2(64) | NO | 统计事件的名称 |
| CLASS | NUMBER | NO | 统计事件所属的类型 |
| VALUE | NUMBER | NO | 统计事件对应的结果数值 |
| STAT_ID | NUMBER | NO | 统计事件 ID |

# **GV$SESSION_EVENT/V$SESSION_EVENT**
## 功能
GV$SESSION_EVENT 展示集群内所有服务器的 Session 级别的等待事件。
V$SESSION_EVENT 展示每一个 Session 发生的等待事件。
## 字段说明
| **字段名称** | **类型** | **是否可以为 NULL** | **描述** |
| --- | --- | --- | --- |
| CON_ID | bigint(20) | NO | 租户 ID |
| SVR_IP | varchar(46) | NO | 服务器 IP 地址 |
| SVR_PORT | bigint(20) | NO | 服务器端口号 |
| SID | bigint(20) | NO | Session ID |
| EVENT | varchar(64) | NO | 等待事件的描述 |
| TOTAL_WAITS | bigint(20) | NO | 等待事件的总等待次数 |
| TOTAL_TIMEOUTS | bigint(20) | NO | 等待事件的总等待超时次数 |
| TIME_WAITED | double | NO | 等待事件的总等待时间，单位为百分之一秒 |
| AVERAGE_WAIT | double | NO | 等待事件的平均等待时间，单位为百分之一秒 |
| MAX_WAIT | double | NO | 等待事件的最长等待时间，单位为百分之一秒 |
| TIME_WAITED_MICRO | bigint(20) | NO | 等待事件的总等待时间（微秒）  |
| CPU | null | NO | 无效列 |
| EVENT_ID | bigint(20) | NO | 等待事件 ID |
| WAIT_CLASS_ID | bigint(20) | NO | 等待事件所属类型的 ID |
| WAIT_CLASS# | bigint(20) | NO | 等待事件所属类型的下标 |
| WAIT_CLASS | varchar(64) | NO | 等待事件所属类型的名称 |

system_trace_level 
=======================================

system_trace_level 用于设置系统追踪日志的日志打印级别。
**说明**



在当前版本中，该配置项已不再使用。


|      **属性**      |                                                                               **描述**                                                                                |
|------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型             | 整型                                                                                                                                                                  |
| 默认值              | 1                                                                                                                                                                   |
| 取值范围             | * 0：表示 none   * 1：表示 standard   * 2：表示 debug    |
| 是否重启 OBServer 生效 | 否                                                                                                                                                                   |

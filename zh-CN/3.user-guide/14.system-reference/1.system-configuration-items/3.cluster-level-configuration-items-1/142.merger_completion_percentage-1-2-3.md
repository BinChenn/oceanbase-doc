merger_completion_percentage 
=================================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/merger_completion_percentage-1-2-3
merger_completion_percentage 用于设置合并完成副本数的百分比阈值，达到该阈值则认为本轮合并完成调度。


|      **属性**      |   **描述**   |
|------------------|------------|
| 参数类型             | 整型         |
| 默认值              | 100        |
| 取值范围             | \[5, 100\] |
| 是否重启 OBServer 生效 | 否          |

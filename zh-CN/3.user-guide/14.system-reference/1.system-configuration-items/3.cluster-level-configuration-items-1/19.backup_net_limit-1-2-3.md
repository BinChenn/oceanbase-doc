backup_net_limit 
=====================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/backup_net_limit-1-2-3
backup_net_limit 用于设置集群备份的总带宽。


|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 容量单位      |
| 默认值              | 0M        |
| 取值范围             | \[0M, +∞) |
| 是否重启 OBServer 生效 | 否         |

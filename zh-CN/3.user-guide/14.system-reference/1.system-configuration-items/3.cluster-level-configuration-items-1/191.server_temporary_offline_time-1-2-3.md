server_temporary_offline_time 
==================================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/server_temporary_offline_time-1-2-3
server_temporary_offline_time 用于设置节点心跳中断的时间阈值，即节点心跳中断多久后认为其被临时下线。

> **说明**
> 
> 在当前版本中，该配置项已不再使用。


|      **属性**      |   **描述**   |
|------------------|------------|
| 参数类型             | 时间类型       |
| 默认值              | 60s        |
| 取值范围             | \[15s, +∞) |
| 是否重启 OBServer 生效 | 否          |

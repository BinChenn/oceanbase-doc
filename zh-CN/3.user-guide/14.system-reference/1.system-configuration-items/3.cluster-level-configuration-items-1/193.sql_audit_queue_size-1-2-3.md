sql_audit_queue_size 
=========================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/sql_audit_queue_size-1-2-3
sql_audit_queue_size 用于设置 SQL 审计数据存储的最大记录条数。

> **说明**
> 
> 在当前版本中，该配置项已不再使用。


|        属性        |      描述      |
|------------------|--------------|
| 参数类型             | 整型           |
| 默认值              | 10000000     |
| 取值范围             | \[10000, +∞) |
| 是否重启 OBServer 生效 | 否            |

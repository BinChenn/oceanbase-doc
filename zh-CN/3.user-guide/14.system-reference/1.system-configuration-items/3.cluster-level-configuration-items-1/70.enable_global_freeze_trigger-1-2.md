enable_global_freeze_trigger 
=================================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/enable_global_freeze_trigger-1-2
enable_global_freeze_trigger 用于设置是否开启自动触发全局冻结。


|      **属性**      |                                                 **描述**                                                 |
|------------------|--------------------------------------------------------------------------------------------------------|
| 参数类型             | 布尔类型                                                                                                   |
| 默认值              | False                                                                                                  |
| 取值范围             | <li> True   <li> False    |
| 是否重启 OBServer 生效 | 否                                                                                                      |




> **说明**
> 
> 该配置项保持保持默认值即可，不推荐使用。

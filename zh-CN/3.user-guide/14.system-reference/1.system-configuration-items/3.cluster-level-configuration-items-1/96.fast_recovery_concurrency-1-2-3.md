fast_recovery_concurrency 
==============================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/fast_recovery_concurrency-1-2-3
`fast_recovery_concurrency` 用于控制 RootService 调度的快速恢复任务在 OBServer 上执行的并发数。


|        属性        |     描述     |
|------------------|------------|
| 参数类型             | 整型         |
| 默认值              | 10         |
| 取值范围             | \[1,1000\] |
| 是否重启 OBServer 生效 | 否          |

# 使用 ODC 创建表、视图、函数、存储过程

OceanBase 开发者中心（OceanBase Developer Center，ODC）支持可视化方式创建表、视图、函数、存储过程等数据库对象。

## 背景信息

在 ODC 上创建 Oracle 模式的表、视图、函数、存储过程的方式与 MySQL 一样。

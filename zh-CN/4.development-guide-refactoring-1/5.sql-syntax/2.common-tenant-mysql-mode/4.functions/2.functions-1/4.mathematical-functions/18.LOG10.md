LOG10 
==========================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/LOG10


声明 
-----------------------

```unknow
LOG10(X)
```



说明 
-----------------------

返回以 10 为底的 `X` 的指数。

示例 
-----------------------

```unknow
obclient> SELECT LOG10(1000);
+-------------+
| LOG10(1000) |
+-------------+
|           3 |
+-------------+
1 row in set (0.01 sec)

obclient> SELECT LOG10(-1000);
+--------------+
| LOG10(-1000) |
+--------------+
|         NULL |
+--------------+
1 row in set, 1 warning (0.00 sec)
```

OLD_CURRENT_SCHEMA 
=======================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/OLD_CURRENT_SCHEMA
`OLD_CURRENT_SCHEMA` 函数用于从 `sys_context` 返回会话值（'`userenv`', '`current_schema`'）。

语法 
-----------------------

```sql
DBMS_UTILITY.OLD_CURRENT_SCHEMA 
RETURN VARCHAR2;
```

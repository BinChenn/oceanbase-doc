CREATEXML 
==============================

CREATEXML 静态函数用于创建并返回一个 XMLType 实例。CLOB 参数必须包含正确的格式和有效的 XML 文档。

语法 
-----------------------

OceanBase 数据库当前版本支持从字符串 和 CLOB 创建 XMLType 实例。语法如下：

```unknow
STATIC FUNCTION CREATEXML(
       xmlData IN VARCHAR2)
RETURN XMLType;

STATIC FUNCTION CREATEXML(
   xmlData IN CLOB)
RETURN XMLType;
```



参数解释 
-------------------------



|   参数    |                   解释                    |
|---------|-----------------------------------------|
| xmlData | `CLOB` 和 `VARCHAR2` 格式或所对应的对象类型格式的实际数据。 |

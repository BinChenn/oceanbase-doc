FREMOVE 
============================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/FREMOVE
`FREMOVE` 存储过程用于删除指定的文件。

语法 
-----------------------

```unknow
UTL_FILE.FREMOVE (
location IN VARCHAR2,
filename IN VARCHAR2);
```



参数解释 
-------------------------



|    参数    |       解释        |
|----------|-----------------|
| location | 文件所在的路径（大小写敏感）。 |
| filename | 需要被删除的文件名。      |



异常情况 
-------------------------

* `ACCESS_DENIED`

  

* `DELETE_FAILED`

  

* `INVALID_FILENAME`

  

* `INVALID_OPERATION`

  

* `INVALID_PATH`

  




使用说明 
-------------------------

`FREMOVE` 在删除文件之前不会验证权限。但操作系统会验证文件和目录权限。

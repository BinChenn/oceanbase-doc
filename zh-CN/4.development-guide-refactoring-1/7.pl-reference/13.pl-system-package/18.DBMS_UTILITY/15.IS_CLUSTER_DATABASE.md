IS_CLUSTER_DATABASE 
========================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/IS_CLUSTER_DATABASE
`IS_CLUSTER_DATABASE` 函数用于确定数据库是否在集群数据库模式下运行。

语法 
-----------------------

```unknow
DBMS_UTILITY.IS_CLUSTER_DATABASE 
RETURN BOOLEAN;
```



返回值 
------------------------

如果此实例以集群数据库模式启动，则此函数返回 `TRUE`；否则返回 `FALSE`。

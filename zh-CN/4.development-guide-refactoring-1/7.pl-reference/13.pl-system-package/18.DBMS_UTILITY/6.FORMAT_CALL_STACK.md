FORMAT_CALL_STACK 
======================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/FORMAT_CALL_STACK
`FORMAT_CALL_STACK` 函数用于格式化当前的调用堆栈。

`FORMAT_CALL_STACK` 适用于访问调用堆栈的任何存储过程或触发器，可用于调试。

语法 
-----------------------

```unknow
DBMS_UTILITY.FORMAT_CALL_STACK 
RETURN VARCHAR2;
```



返回值 
------------------------

返回调用堆栈，最多 2000 个字节。

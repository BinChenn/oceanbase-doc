GETTYPE 成员函数 
=================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/gettype-member-function
`GETTYPE` 成员函数用于获取 `ANYDATA` 的类型代码。

语法 
-----------------------

```unknow
MEMBER FUNCTION GetType(
         self IN AnyData, 
         typ OUT NOCOPY AnyType)
RETURN PLS_INTEGER；
```



参数解释 
-------------------------



|  参数  |             描述             |
|------|----------------------------|
| self | 一个 `ANYDATA`。              |
| typ  | 对应于 `ANYDATA` 的 `AnyType`。 |



返回值 
------------------------

与 `ANYDATA` 类型相对应的类型。

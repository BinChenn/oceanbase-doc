CLOSE_CURSOR 
=================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/CLOSE_CURSOR
CLOSE_CURSOR 存储过程用于关闭指定的游标。

语法 
-----------

```sql
DBMS_SQL.CLOSE_CURSOR(cursor_id    IN INTEGER);
```



参数 
-----------



|  **参数**   |    **解释**    |
|-----------|--------------|
| cursor_id | 要关闭的游标 ID 号。 |

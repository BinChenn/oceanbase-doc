# 用户被锁报错

本文为您介绍用户被锁的相关报错及处理方式。

## 错误：ERROR 3118

### 报错信息

```unknow
ERROR 3118 (HY000):User locked
```

### 示例

登录 `ny1` 提示用户被锁。

```unknow
[admin@k08j13249.eu95sqa /home/admin]
$obclient -h10.0.0.0 -P2881 -uny@Oracle -p -A
Enter password:
ERROR 3118 (HY000): User locked
```

1. 登录 `sys` 用户。

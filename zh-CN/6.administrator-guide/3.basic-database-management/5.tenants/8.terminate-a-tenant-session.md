终止租户会话 
===========================

OceanBase 数据库支持通过 `KILL` 语句终止租户会话。

终止会话的 SQL 语句如下：

```sql
KILL [CONNECTION] 'session_id'
```


**说明**



如果您拥有 `PROCESS` 权限，则您可以查看所有会话。如果您拥有 `SUPER` 权限，您可以终止所有会话和语句。否则，您只能查看和终止您自己的会话和语句。

示例：

```sql
obclient> KILL session_id;
```



或者

```sql
obclient> KILL CONNECTION session_id;
```

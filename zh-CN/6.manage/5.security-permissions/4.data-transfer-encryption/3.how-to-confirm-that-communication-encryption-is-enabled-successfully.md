# 如何确定通信加密开启成功

## 检查 MySQL 端口是否开启 SSL

通过 MySQL/OBClient 登录，执行`\s`查看 SSL 字段的示例如下：

```sql
obclient> \s
```

## 检查 RPC 端口是否开启 SSL

在 `OBServer.log` 中的日志，搜索关键字 "RPC connection accept" 查看 use_ssl = True 还是 False 判定是否 RPC SSL 开启成功。

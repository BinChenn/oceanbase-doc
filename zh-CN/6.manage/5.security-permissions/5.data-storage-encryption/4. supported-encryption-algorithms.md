# 支持的加密算法

OceanBase 数据库现阶段支持国际上主流的加密算法 AES，与此同时，还支持国密算法 SM4。

| 算法  |                            密钥长度                            | 模式  |
|-----|------------------------------------------------------------|-----|
| AES | 128 bits 192 bits 256 bits | ecb |
| SM4 | 128 bits                                                   | cbc |

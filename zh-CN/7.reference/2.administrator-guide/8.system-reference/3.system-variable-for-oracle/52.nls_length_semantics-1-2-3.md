nls_length_semantics 
=========================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/nls_length_semantics-1-2-3
nls_length_semantics 表示 char、varchar2 类型 的 length 语义。


|  **属性**  |                                                   **描述**                                                   |
|----------|------------------------------------------------------------------------------------------------------------|
| 参数类型     | varchar                                                                                                    |
| 默认值      | BYTE                                                                                                       |
| 取值范围     | <li> BYTE   <li> CHAR         |
| 生效范围     | <li> GLOBAL   <li> SESSION    |
| 是否参与序列化  | 是                                                                                                          |
| 是否影响计划生成 | 是                                                                                                          |

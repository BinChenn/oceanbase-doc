TERMINATE 
==============================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/TERMINATE
用户调用 TERMINATE 函数用以终止对包的使用。

>**功能适用性**
>
>该内容仅适用于 OceanBase 数据库企业版。OceanBase 数据库社区版仅提供 MySQL 模式。

语法 
-----------

```sql
DBMS_RANDOM.TERMINATE
```



使用说明 
-------------

此程序不执行任何功能，尽管当前支持此程序，但已过期，不建议使用。

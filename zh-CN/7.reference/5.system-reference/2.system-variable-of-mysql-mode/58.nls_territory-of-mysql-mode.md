# nls_territory

`nls_territory` 表示数据库国际化支持中的当前地域，与 nls_language 结合使用。

> **说明**
>
> 该变量当前暂不生效。

| **属性**  |                                                   **描述**                                                   |
|---------|------------------------------------------------------------------------------------------------------------|
| 参数类型    | varchar                                                                                                    |
| 默认值     | AMERICA                                                                                                    |
| 取值范围    | N/A                                                                                                        |
| 生效范围    | <li> Global   <li> Session    |
| 是否参与序列化 | 是                                                                                                          |

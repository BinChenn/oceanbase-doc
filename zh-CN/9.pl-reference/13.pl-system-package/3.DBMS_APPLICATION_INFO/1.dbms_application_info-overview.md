DBMS_APPLICATION_INFO 概述 
=============================================

DBMS_APPLICATION_INFO 系统包用于将应用程序名称注册在数据库中以进行审计或性能跟踪。

DBMS_APPLICATION_INFO 权限设置 
-----------------------------------------------

使用 `DBMS_APPLICATION_INFO` 系统包无需特殊权限。当应用程序向数据库注册时，它的名称和操作记录在 `__all_virtual_processlist` 视图中。
